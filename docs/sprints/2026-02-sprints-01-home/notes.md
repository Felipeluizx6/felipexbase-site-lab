# Sprint 01 — Home (Fevereiro/2026)

## Objetivo
Documentar a versão inicial da Home e identificar pontos de melhoria.

## Estado Atual
- Layout visual forte
- Excesso de informação
- CTA pouco hierarquizado
- Tipografia pesada
- Fundo poluído visualmente

## Problemas Identificados
- Baixa legibilidade
- Falta de formulário
- Falta de funil estruturado
- Elementos sem função real
- Imagens apenas decorativas

## Decisões
- Refatorar layout com containers em camadas
- Simplificar copy
- Implementar formulário
- Preparar integração com CRM
- Trabalhar SEO técnico

## Próximos Passos
- Criar seção de captura
- Ajustar hierarquia visual
- Implementar segurança
- Otimizar mobile
- Medir conversão
